Pinterest Video Downloader

A web application to download videos from Pinterest

Project Information:

This is a Flask web application that allows you to download videos from Pinterest links.

The application provides both a user-friendly web interface and an API for developers.

Key Features:
  • Download videos from both Pinterest URLs and pin.it short links
  • Get videos in the highest available quality
  • Simple and intuitive user interface
  • REST API for programmatic access
  • Docker support for easy deployment
Deployment Options:

This application needs to be deployed to a server that can run Python/Flask applications. GitHub Pages can only host static sites, not dynamic Flask applications.

Recommended Deployment Options:
  • Heroku: Simple deployment with free tier available
  • PythonAnywhere: Specialized hosting for Python applications
  • Google Cloud Run: Containerized deployment with Docker
  • AWS Elastic Beanstalk: Managed service for web applications

Check the repository README for detailed deployment instructions.
